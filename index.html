Laravel Sekarang
Laravel ialah rangka kerja (framework) PHP yang memudahkan pembangunan aplikasi web dengan menyediakan ciri seperti routing, pengurusan database, dan sistem pengesahan secara mudah dan moden.
Html & CSS Sekarang
HTML (HyperText Markup Language) digunakan untuk membina struktur asas laman web, manakala CSS (Cascading Style Sheets) digunakan untuk mencantikkan dan menggayakan laman web tersebut.
PHP Sekarang
PHP (Hypertext Preprocessor) ialah bahasa pengaturcaraan pelayan yang digunakan untuk membina laman web dinamik dan aplikasi web. Ia mudah dipelajari, fleksibel, dan sering digunakan bersama pangkalan data seperti MySQL.
MySQL Sekarang
MySQL ialah sistem pengurusan pangkalan data berasaskan SQL yang digunakan untuk menyimpan, mengurus, dan mengakses data secara cekap dalam aplikasi web dan perisian.
Git & GitHub Sekarang
Git ialah sistem kawalan versi yang membolehkan anda menjejak perubahan kod dan bekerjasama dengan pasukan. GitHub pula ialah platform berasaskan web untuk menyimpan repositori Git, berkongsi kod, dan bekerjasama dalam projek perisian secara dalam talian.
